##### Input data

This tab gives you an overview of model input data.

First you get an overview of data that is country specific: Weighted Average Cost of Capital (WACC); natural gas production costs, natural gas price; natural gas production losses

Next follows a list of global input data (that is not country specific). It is structured by expandable sections, each one containing a data table.

You can edit all data tables in this tab when activating data edit mode (at the bottom of the sidebar).
